Installation on Shared Hosting :

Step One - cPanel and FTP

You need to have cPanel or FTP access to install on shared hosting . Create a new FTP account or if you have existing download filezilla config file , install Filezilla and import new config and connect to server using filezilla , now upload all files into your website folder, extract it and open cpanel.
Or Directly enter cPanel >> Filemanager >> PUBLIC_HTML >> upload all files into your website folder, and extract it .

Useful Tip:

If you want to install this app on your root directory i.e Yourwebsite.com Then move all files from RMS folder to the root directory. OR If you are intersted in installing this app on yourwebsite.com/directory then follow my steps which is mentioned in the gif. If there is any problem while configuring shoot an email to me i will configure it for you, iftipro@gmail.com

Step Two - Database and Configure .env

After opening cpanel go to mysql databases and create new database , then create new user , then from mysql databases screen connect user and database and grant all permissions .

Database Creation

Configuring .env

Now go to the root directory of your application and change username and password in .env file to the one you created.

Step Three - Import Database File

You will need to import the database file which will be in importantfiles directory of your app which you have downloaded. The file will have .sql extension. Go to PHPMyadmin >> Select the database your have created for your app, and click on Import >> Select the database file and click Go !

How to search for Student Result

All you need to search for result, is CLICK on the BIG blue RESULT Button and you will meet with a search box enter your roll no and hit go !

Useful Tip:

By Clicking on Generate PDF you will get your Result Card in PDF Form
